# 🎰 台灣抽卡機地圖

社群共建的台灣 IP 抽卡機 / 快閃活動地圖，資料由管理者維護於 Google Sheet，網站自動讀取並顯示於地圖上。

**🔗 [查看網站](https://romiajoin.github.io/taiwan-gacha-map)**  
**最後更新：** 2026/06/01

---

## 功能

- 🗺️ 互動地圖，顯示全台抽卡機與 IP 快閃活動地點
- 🔍 搜尋 IP 角色、地點名稱
- 📍 點地標查看詳細資訊（場地、IP、彈數、期間限定、圖片）
- 🖼️ 支援多張圖片輪播，點擊放大
- 📱 手機版響應式設計（上方地圖 + 下方列表）
- 🗺️ 一鍵導航至 Google Maps
- 📅 依期間限定結束日期排序

---

## 技術架構

| 項目 | 工具 |
|------|------|
| 前端 | 純 HTML / CSS / JavaScript（單一 index.html） |
| 地圖套件 | Leaflet.js 1.9.4（OpenStreetMap 底圖，免費） |
| 資料來源 | Google Sheet（發布為公開 CSV） |
| 圖片託管 | Cloudinary |
| 網站託管 | GitHub Pages（免費） |
| 字體 | Noto Sans TC、Space Mono（Google Fonts） |

**不需要後端、不需要資料庫、不需要 API 金鑰。**

## Google Sheet 欄位規格

| 欄 | 欄位 | 說明 | 必填 |
|----|------|------|------|
| A | 店名 | 活動或地點名稱 | ✅ |
| B | 場地 | 所在建築或商場（如：三創生活 7F） | ❌ |
| C | 縣市 | 縣市名稱（如：台北市） | ❌ |
| D | 地址 | 詳細地址（含縣市） | ✅ |
| E | 緯度 | 數字，用於地圖定位 | ✅ |
| F | 經度 | 數字，用於地圖定位 | ✅ |
| G | 期間限定 | 活動日期（如：2026/05/29～2026/07/23） | ❌ |
| H | IP | 熱門角色或 IP 名稱 | ❌ |
| I | 圖片 | Cloudinary 圖片網址（多張用逗號分隔） | ❌ |
| J | 彈數 | 如：第一彈,第二彈 | ❌ |
| K | 一抽張數 | 如：2張 | ❌ |
| L | 備註 | 補充說明 | ❌ |

> 緯度經度可以用 Google Maps 點地點後取得。  
> 欄位為空時，對應資訊不顯示，不影響版面。

## 彈窗顯示內容

點地標後顯示（有資料才顯示）：

1. 店名（黃色標題）
2. 🏢 場地
3. 📍 地址
4. ⭐ IP
5. 🃏 彈數
6. 🎴 一抽張數
7. 📝 備註
8. ⏰ 期間限定（黃色圓角框）
9. 圖片（支援輪播，點擊放大）
10. 在 Google Maps 查看 →

## 資料更新流程

1. 在 Google Sheet 新增或編輯資料
2. 網站重新整理後自動讀取，不需修改程式碼

### 新增圖片
1. 上傳圖片到 Cloudinary
2. 複製圖片網址
3. 貼到 Google Sheet I 欄（多張用逗號分隔）

### 程式碼更新
```bash
./push.sh "說明改了什麼"
```
